Styling Text and Fonts in CSS CSS allows for extensive customization of text and fonts on your webpage. You can define the appearance of text, including its font family, size, weight, alignment, and much more. This control gives you the flexibility to create visually appealing and readable content. Common CSS Text Properties: font-family: Specifies the font to be used for text. font-size: Determines the size of the text. font-weight: Defines the thickness of the text. text-align: Aligns text to the left, right, center, or justify. line-height: Sets the space between lines of text. color: Changes the color of the text. Explanation of the CSS Styles: Body: font-family: Arial, sans-serif;: Sets the default font for the entire page to Arial, with a fallback to any sans-serif font. font-size: 16px;: Defines the default font size as 16 pixels. line-height: 1.6;: Sets the line height, making the text easier to read by adding space between lines. color: #333;: Changes the default text color to dark gray. Headings: h1: font-family: 'Georgia', serif;: Changes the font family to Georgia, with a fallback to any serif font. font-size: 36px;: Increases the font size to 36 pixels for the main heading. font-weight: bold;: Makes the text bold. text-align: center;: Centers the heading text. color: #1a73e8;: Changes the text color to a blue shade. h2: font-family: 'Tahoma', sans-serif;: Uses the Tahoma font for the subheading. font-size: 28px;: Sets the font size to 28 pixels. text-align: left;: Aligns the text to the left. color: #ff5722;: Applies an orange color to the subheading text. Paragraphs: p: font-family: 'Verdana', sans-serif;: Sets the font to Verdana. font-size: 18px;: Increases the font size for paragraphs. text-align: justify;: Justifies the paragraph text to align both left and right edges. color: #4a4a4a;: Changes the paragraph text color to grayish. Highlighted Paragraph: .highlight: font-weight: bold;: Makes the text bold. font-size: 20px;: Increases the font size. color: #e91e63;: Changes the text color to pink. Links: a: font-family: 'Courier New', monospace;: Applies a monospace font to links. font-size: 16px;: Sets the link text size. font-weight: bold;: Makes the link text bold. color: #007bff;: Sets the link color to blue. text-decoration: none;: Removes the default underline from the link. Conclusion: CSS provides various properties for customizing text and fonts, allowing developers to create visually appealing web pages. By controlling properties like font-family, font-size, font-weight, and text-align, you can make text more readable and aligned with the design of your site.

Alignment and Display in CSS Text Alignment text-align: Used to align text within its container. left: Aligns text to the left. center: Centers the text. right: Aligns text to the right. justify: Justifies the text, spreading it out evenly across the line. Vertical Alignment vertical-align: Used to align the content of an inline element vertically within its line box. top: Aligns the top of the element with the top of the line box. middle: Aligns the middle of the element with the middle of the line box. bottom: Aligns the bottom of the element with the bottom of the line box. baseline: Aligns the baseline of the element with the baseline of the other elements on the line. Display Properties block: Elements are displayed as blocks, taking up the full width of their container and starting on a new line. inline: Elements are displayed inline, without starting on a new line. inline-block: Elements are displayed inline, but they can have width, height, and margins like block elements. flex: Enables flexible box layout, providing a powerful way to arrange elements and control their spacing.
